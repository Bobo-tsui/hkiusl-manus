## 構思一：「科技浪潮」— 動態流體設計

**設計運動**: Fluid Design / Organic Modernism
**核心原則**:
1. 以流動的波浪形態象徵科技創新的浪潮
2. 大面積留白搭配重點色塊形成視覺節奏
3. 非對稱佈局打破傳統網頁的呆板感
4. 以動態元素引導用戶視線流動

**色彩哲學**: 以午夜藍（#1a1a4e）為基底象徵深度與專業，薰衣草紫（#b8a9d4）作為活力點綴，奶油白（#faf8f5）提供呼吸空間。三色交織如同科技與人文的融合。

**佈局範式**: 全寬流體佈局，每個區塊以波浪形 SVG 分隔線銜接，形成連綿不斷的視覺流動。Hero 區域佔據全屏，內容區塊交替使用左右偏移的不對稱排列。

**標誌性元素**:
- 流動的漸層波浪作為區塊分隔
- 浮動的半透明圓形裝飾元素

**互動哲學**: 滾動時波浪元素產生視差效果，卡片懸停時輕微上浮並投射柔和陰影。

**動畫**: 入場動畫使用 stagger fade-in，波浪分隔線隨滾動緩慢流動，按鈕懸停時產生漣漪效果。

**字體系統**: 標題使用 Noto Sans TC Bold（700），正文使用 Noto Sans TC Regular（400），數字使用 Space Grotesk 增添科技感。

## 構思二：「稜鏡折射」— 玻璃態設計

**設計運動**: Glassmorphism + Neo-brutalism 混搭
**核心原則**:
1. 毛玻璃效果的卡片層疊營造深度感
2. 大膽的幾何色塊作為背景裝飾
3. 粗體標題與纖細正文形成強烈對比
4. 網格系統但允許元素「突破」邊界

**色彩哲學**: 深邃午夜藍（#0d1b3e）為主背景，薰衣草漸層（#c4b5e0 → #e8dff5）作為玻璃卡片的底色，奶油金（#f5f0e8）用於重點文字。色彩如同光線穿過稜鏡般折射出不同層次。

**佈局範式**: 基於 CSS Grid 的模組化佈局，卡片以不同大小和層次堆疊。Hero 區域使用大型毛玻璃面板覆蓋在漸層背景上，內容區塊以交錯的網格呈現。

**標誌性元素**:
- 毛玻璃效果的浮動卡片
- 幾何形狀的漸層裝飾（三角形、圓形）

**互動哲學**: 卡片懸停時玻璃效果加深，背景裝飾隨滑鼠微移，營造立體空間感。

**動畫**: 元素以 scale + fade 方式進場，卡片懸停時產生 glassmorphism 深度變化，滾動觸發的漸進式顯現。

**字體系統**: 標題使用 Noto Sans TC Black（900），正文使用 Noto Sans TC Light（300），英文使用 Outfit 字體。

## 構思三：「墨韻新章」— 東方現代主義

**設計運動**: Eastern Modernism / Zen Minimalism
**核心原則**:
1. 以書法筆觸的動態線條作為裝飾元素
2. 大量留白（「空」的美學）與精準的內容放置
3. 垂直與水平的節奏交替，如同中式卷軸的展開
4. 現代科技感與東方美學的融合

**色彩哲學**: 以墨色午夜藍（#141e3a）為底蘊，薰衣草紫（#a89cc8）如同淡墨暈染，奶油宣紙色（#f8f4ec）作為主要背景。色彩搭配如同水墨畫的濃淡乾濕。

**佈局範式**: 卷軸式單欄佈局為主，關鍵區塊使用雙欄不對稱排列。Hero 區域以全屏圖片搭配大型標題，日程表以時間軸形式垂直展開，合作機構以橫向滾動展示。

**標誌性元素**:
- 抽象的墨跡/筆觸裝飾線條
- 圓形「印章」風格的圖標設計

**互動哲學**: 滾動如同展開卷軸，內容逐步顯現。懸停效果含蓄優雅，如同墨跡暈開。

**動畫**: 內容以「書寫」般的動態從左至右顯現，區塊以淡入方式出現，時間軸節點依序亮起。

**字體系統**: 標題使用 Noto Serif TC Bold 增添文化底蘊，正文使用 Noto Sans TC Regular 保持現代可讀性，英文使用 DM Sans。

## 最終選擇：構思一「科技浪潮」— 動態流體設計

選擇理由：此設計最能體現活動的科技創新精神與跨校合作的流動性。流體設計語言既現代又不失親和力，適合面向大專學生的活動網頁。午夜藍、薰衣草紫、奶油白的配色方案完美契合用戶要求的 lavender, cream, midnight blue 色調。
